# Carbohydrate-Calculator-Test-Scripts
Personally made Test Scripts using Selenium in Java

To execute use command: "mvn clean test"

Note: Test Case 3 (Stress Test) loop duration can be modified (default 10). To modify simply use command:\
"mvn clean test -DstressLoops=(# of loops)"\
\
Example: "mvn clean test -DstressLoops=20"\
This will set Stress Test to loop 20 times instead of 10
